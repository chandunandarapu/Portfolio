🌐 Personal Portfolio & Blog Website

A modern developer portfolio and blog website built using HTML, Tailwind CSS, and JavaScript to showcase projects, skills, services, and professional experience.

This website highlights my work in web development, data analytics, and backend systems while also providing a blog and service section for sharing insights and offering digital solutions.

👨‍💻 About the Project

This project is a personal portfolio platform designed to:

Showcase technical skills and projects

Present professional journey and experience

Share insights through blog content

Offer web development and data services

Provide an easy way for clients or recruiters to contact me

The website includes two main pages:

1️⃣ Portfolio Website – Personal profile, projects, skills, and contact information
2️⃣ Blog & Services Page – Articles, services offered, testimonials, and business information
